# 🐍 스네이크 게임 (Snake Game)

단일 HTML 파일로 실행되는 간단하고 직관적인 **스네이크 게임**입니다.  
화살표 키로 조작하며, **점수 / 최고 점수 / 속도 증가 / 일시정지 / 재시작** 기능을 포함합니다.

---

## 🚀 실행 방법

1. 저장된 `snake.html` 파일을 브라우저에서 실행합니다.
2. 별도의 설치나 빌드 과정 없이 바로 플레이할 수 있습니다.
3. PC 환경(키보드 사용)을 권장합니다.

---

## 🎮 조작법

| 기능 | 조작 |
|------|------|
| 이동 | ↑ ↓ ← → |
| 다시 시작 | `R` |
| 일시정지 / 재개 | `Space` |

---

## ⭐ 주요 기능

- **단일 HTML 파일**로 구성 (HTML + CSS + JavaScript 포함)
- **Canvas 기반 그래픽**
- **로컬 스토리지(LocalStorage)** 를 이용한 최고 점수 저장
- 점수를 먹을수록 **속도 자동 증가**
- **일시정지 / 재개 기능**
- 반응형 인터페이스(UI)
